Project Calculator

This is the most significant project I have completed using Python, featuring a graphical user interface built with Tkinter. The project serves as a major milestone in my journey, with the help of one and only BRO CODE when doing this project. It represents a stepping stone in my growth, inspiring me to create even more complex and impactful applications in the future.

Project Tic-Tac-Toe

Following the Project Calculator, I completed the Tic-Tac-Toe game, my second project using Tkinter. This project deepened my understanding of GUI development and inspired me to take on more challenges as I continue to grow in programming.

Project Snake Game

The Snake Game was the final project offered by Bro Code, and it posed one of the most challenging problems I’ve faced so far. The difficulty lay in calculating the precise points on the grid where the snake would move next. It took me three days of persistence and countless retries—especially when the snake escaped through the window—but this experience taught me a valuable lesson: success requires patience, perseverance, and embracing the struggles along the way.
